Was ist TfCD?

TfCD steht für "Time for CD" und ist eine Bezeichnung für eine im Internet populäre rechtliche Vereinbarung zwischen Model und Fotograf. Der Grundgedanke von TfCD besteht darin, dass der Fotograf kein Honorar für seine Arbeit erhält. Das Model erhält als Honorar eine CD/DVD mit den Fotos vom Shooting. Diese Fotos können dann auf einer Internetseite angzeigt, ausgedruckt oder für eine Model-Mappe verwendet werden. Der Fotograf darf die Fotos zur Eigenwerbung nutzen. Model und Fotograf unterschreiben vor dem Shooting einen TfCD Vertrag der alle Einzelheiten klärt. Die kommerzielle Nutzung dieser Fotos hingegen ist sowohl Model als auch Fotograf untersagt.
